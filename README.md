# Supreme-Drop-Bot
A supreme bot, written in python, to grab a list of specified products before they sell out!

## Use/Install

### Windows
1. Download and unzip latest build.
2. Run main.exe

### Linux/Other
1. Download source.
1. Install all dependencies with pip.
2. Run main.py
